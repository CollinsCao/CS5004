import java.util.Arrays;
public class Student implements Comparable {

	private String lastName;
	private int studentID;

	public Student() {
		this.lastName = "";
		this.studentID = 0;
	}

	public Student(String lastName, int studentID) {
		this.lastName = lastName;
		this.studentID = studentID;
	}

	public String getLastName() { return lastName; }
	public int getStudentID() { return studentID; }

	public void setLastName(String lastName) { this.lastName = lastName; }
	public void setStudentID(int id) { this.studentID = id; }

// ---- use ID to sort ----
	
	@Override
	public int compareTo(Object other) {
		Student otherStudent = (Student) other;
		if (this.studentID < otherStudent.studentID) return -1;
		if (this.studentID > otherStudent.studentID) return 1;
		return 0;
	}

// 	use compareTo change to lastName sort（
 	public int compareTo(Object other) {
		Student otherStudent = (Student) other;
		return this.lastName.compareTo(otherStudent.lastName);
	}
	
	@Override
	public String toString() {
		return "Student[ID=" + studentID + ", lastName=" + lastName + "]";
	}

	public static void main(String[] args) {
		Student[] students = new Student[5];
		students[0] = new Student("Wang", 3);

		students[3] = new Student("Adams", 2);
		students[4] = new Student("Brown", 4);
	
	Arrays.sort(students); 
	for (Student s: students) {

Class: Movie
1. Abstract or Concrete?
Concrete class — there is no abstract keyword before the class declaration.
2. Superclass and Subclasses:

Superclass: Object (all Java classes implicitly extend Object)
Subclasses: ActionMovie, ComedyMovie, DramaMovie

3. Variables:
MovieID

Purpose: Stores the ID number of the movie
Access level: private — only accessible within the Movie class

title

Purpose: Stores the title of the movie
Access level: private — only accessible within the Movie class

rating

Purpose: Stores the rating of the movie (e.g. R, PG-13)
Access level: private — only accessible within the Movie class

4. Methods:
Movie()

Input: Takes no input
What it does: Default constructor, initializes MovieID to 0, title to "", and rating to ""
Returns: Nothing (void)

Movie(int ID, String title, String rating)

Input: Takes in an int called ID, a String called title, and a String called rating
What it does: Parameterized constructor, sets all three instance variables using the given values
Returns: Nothing (void)

getID()

Input: Takes no input
What it does: Returns the MovieID of this movie
Returns: an int

setID(int ID)

Input: Takes in an int called ID
What it does: Sets the MovieID to the given value
Returns: Nothing (void)

getTitle()

Input: Takes no input
What it does: Returns the title of this movie
Returns: a String

setTitle(String name)

Input: Takes in a String called name
What it does: Sets the title to the given value
Returns: Nothing (void)

getRating()

Input: Takes no input
What it does: Returns the rating of this movie
Returns: a String

setRating(String rating)

Input: Takes in a String called rating
What it does: Sets the rating to the given value
Returns: Nothing (void)

equals(Object other)

Input: Takes in an Object called other
What it does:

If other is null → returns false
If the two objects are not the same class → returns false
If both objects have the same MovieID → returns true
Otherwise → returns false


Returns: a boolean (true if equal, false otherwise)

calcLateFees(int daysLate)

Input: Takes in an int called daysLate

class Rental {

    private Movie movie;
    private int daysRented;
    private int daysLate;

    public Rental(Movie movie, int daysRented) {
        this.movie = movie;
        this.daysRented = daysRented;
        this.daysLate = 0; 
    }


    public Movie getMovie()        { return movie; }
    public int getDaysRented()     { return daysRented; }
    public int getDaysLate()       { return daysLate; }


    public void setDaysLate(int daysLate) {
        this.daysLate = daysLate;
    }



    public double getLateFees() {
        return movie.calcLateFees(daysLate);
    }
}
