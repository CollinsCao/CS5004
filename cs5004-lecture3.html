Lecture Notes (Original)

An array is a data structure used to process a collection of data that is all of the same type.

Arrays are objects and array variables hold references.

A derived class inherits the instance variables and methods of its base class.

Examples & Explanation

为什么 arrays 是 objects？

因为数组通过 new 创建，并且变量里存的是引用地址。

为什么 equals 要用 Object？

否则不会真正 override Object.equals()，会导致比较行为错误。
